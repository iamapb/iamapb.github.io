微信小程序 scroll-view scroll-x 不起作用修改

html样式

<view> 
    <scroll-view scroll-x class="wrap_box">
        <view class="wrap">
        <view class="box">1</view>
        <view class="box">1</view>
        <view class="box">1</view>
        <view class="box">1</view>
        <view class="box">1</view>
        </view>
    </scroll-view>
</view>

css样式 (只要把scroll-view设置white-space：nowrape，内部需要拖动的元素设置为inline-block即可)

.wrap_box {
     white-space: nowrap;
    }
.box {
    display: inline-block;
    }
